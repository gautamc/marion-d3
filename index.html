Marion County, Florida, USA - Visualizing distribution of Sexual Offenders/Predators.

This is a visualization of the number of known Sexual Offenders/Predators by zip code in Marion County, Florida. A draker shade means a higher number of known Sexual Offenders/Predators. Click on a region or select the ZIP code from the list given below the map, to display the list of known offenders under the map. The data for this visualization was obtained from the website of the Florida Dept of Law Enforcement on Jun 22 2013. The code can be found on github: https://github.com/gautamc/marion-d3
